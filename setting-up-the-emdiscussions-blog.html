Setting up the EMDiscussions Blog




Up to now, I've posted any email-related 'stuff' on my personal blog, but have now decided to post that content on a separate blog.

So, here it is!



The plumbing!


This is the combination that enables the creation of the blog:

Bluefish - The blog is created with Bluefish - a fast, lightweight, and open-source code editor, using my (admittedly basic) knowledge of good old HTML!

ImageKit - The blog images are hosted by ImageKit, an 'all-in-one media management and delivery platform', They operate a distributed processing network across multiple regions, including Frankfurt.
